# Linear Search

## Description

Linear Search é um algoritmo que percorre uma lista de elementos, um a um, até encontrar o elemento desejado ou até o final da lista.

## Complexity

| Name           | Best            | Average             | Worst               | Memory    | Stable    |
| -------------- | :-------------: | :-----------------: | :-----------------: | :-------: | :-------: |
| **Linear**     | n               | n                   | n                   | 1         | Yes       |
